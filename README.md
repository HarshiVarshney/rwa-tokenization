# 🏢 Brick 
## Real Estate Tokenization dApp
Brick is a Web3-based decentralized application that enables **fractional ownership of real estate** using blockchain technology.

It allows users to:
- Invest in property using tokens
- Earn rental income automatically
- Participate in governance
- Redeem full ownership of flats

---

## 🚀 Features

### 🔐 Wallet Integration
- Connect wallet using MetaMask (ethers.js)
- Real-time wallet connection status

### 💰 Tokenization Model
- 110 tokens represent full building ownership
- Each token = fractional ownership share
- Investors can buy tokens easily

### 📊 Dashboard
- View:
  - Brick Tokens Owned
  - Flats Owned
  - Rent Earned
- Clean Web3 UI with animated gradients

### 🏢 Fractional Ownership
- Buy tokens to own part of the building
- Earn proportional rental income

### 🔁 Rent Distribution
- Smart contract distributes rent automatically
- Based on token ownership %

### 🏠 Flat Redemption
- Tokens are burned on redemption
- Ownership is transferred

### 🗳 Governance
Token holders can vote on:
- Selling the property
- Renovation decisions

---

## 🧠 How It Works

### 1️⃣ Legal Structure
- Property owned by an **SPV (Special Purpose Vehicle)**
- Tokens represent shares in the SPV

### 2️⃣ Token Economics
- Total Supply: 110 tokens
- Price per token: 0.0005eth

### 3️⃣ Income Distribution
- Rent collected → distributed via smart contract
- Example:
  - Total rent = ₹10,000
  - You own 1% ownership → You receive ₹100

### 4️⃣ Redemption Logic
- 10 tokens required to claim 1 2BHK flat
- 12 tokens required to claim 1 3BHK flat
- Tokens are:
  - Burned
  - Converted into NFT (ownership proof)
